A full-stack web application developed using **Flask, SQLite, and Bootstrap** to efficiently manage student records with a clean and professional user interface.

---

📌 Project Overview

The Student Management System is a CRUD-based web application that allows users to:

- Add new students
- View registered students
- Edit student details
- Delete student records

The project demonstrates backend development, database integration, and responsive frontend design.

---

🚀 Features

- ➕ Add Student Records
- 📋 View All Students in Table Format
- ✏️ Edit Student Details
- ❌ Delete Student Records
- 💾 SQLite Database Integration
- 🎨 Professional Bootstrap UI
- 📱 Fully Responsive Design
